\chapter{Introduction}
\label{ch:introduction}

\section{Background and Problem Statement}
In the modern cybersecurity landscape, Security Operations Centers (SOCs) are grappling with a fundamental issue: \textbf{Alert Fatigue}. First-line defense systems (IDS/IPS, Firewalls) continuously generate hundreds of thousands of alerts daily. Due to their reliance on static rules or signatures, these systems lack contextual reasoning capabilities, resulting in an exceptionally high rate of false positives.

Large Language Models (LLMs) offer immense potential for contextual analysis, but deploying them directly into a network pipeline faces three critical barriers:
\begin{enumerate}
    \item \textbf{Prohibitive Latency:} Using LLMs to analyze every high-speed network log in real-time is computationally infeasible.
    \item \textbf{Manipulation Risks (Prompt Injection):} Attackers can inject malicious instructions directly into the payload (e.g., HTTP User-Agent) to deceive the LLM and bypass detection.
    \item \textbf{Data Sovereignty:} Organizations cannot transmit sensitive network logs to third-party APIs (such as cloud services) due to strict Zero-Trust security requirements.
\end{enumerate}

\section{Solution: The SENTINEL Two-Tier Architecture}
To decisively address these challenges, this thesis proposes \textbf{SENTINEL} -- a two-tier cognitive architecture combining a high-speed filter with a locally deployed Agentic AI:
\begin{itemize}
    \item \textbf{Tier-1 (Statistical Filter \& ML Gateway):} Processes massive traffic volumes at wire speed. The ML Gateway (LightGBM) automatically filters out benign data and blocks overt attacks. Only a small minority of ambiguous'' cases are escalated to Tier-2. Additionally, a \textbf{Semantic Cache (Tier 1.75)} instantly blocks repeated attacks (e.g., DDoS, Brute-force) within 1 millisecond without invoking the LLM.
    \item \textbf{Tier-2 (Cognitive LLM Agent):} Employs a locally hosted, quantized LLM (Gemma-9B). It leverages a RAG (Retrieval-Augmented Generation) knowledge base, grounded in the MITRE ATT\&CK framework, to evaluate evidence and issue verdicts. Crucially, the LLM does not autonomously dictate actions; its outputs are constrained by a \textbf{Unified Decision Policy}, ensuring transparency and predictability.
\end{itemize}

This approach demonstrates that integrating deep AI reasoning into network defense is highly practical, cost-effective, resilient against evidence manipulation, and strictly adheres to data privacy constraints.
