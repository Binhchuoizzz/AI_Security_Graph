% Abstract Page
\chapter*{Abstract}
\addcontentsline{toc}{chapter}{Abstract}
\begin{singlespace} % Abstract text is typically single spaced in templates
This thesis proposes a novel two-tier cognitive architecture, named SENTINEL, to automate network security threat triage and response. The first tier utilizes Welford's online statistical anomaly detection to filter out benign traffic at wire speed. The second tier acts as an AI Agent built on LangGraph, utilizing offline large language model (LLM) reasoning and hybrid Dual-RAG (MITRE ATT\&CK and NIST SP 800-61r2) for threat classification and mitigation instruction. Prompt safety is enforced via Delimited Data Encapsulation, knowledge base integrity is cryptographically validated using SHA-256 checksum signing, semantic cache poisoning is mitigated through context-level structural sanitization, and database audit logs are secured using HMAC log chaining. Evaluated on CSE-CIC-IDS2018 and DAPT2020 datasets, the system demonstrates high precision, low latency, and robustness against adversarial attacks.
\end{singlespace}
